# 🧠 Quiz App

Quiz App is a Flutter-based interactive quiz application where users can attempt multiple-choice questions stored in Firebase. After completing the quiz, the app displays the user’s score along with the correct answers for each question.

---

## 📱 Features

- ❓ Multiple-choice quiz system
- ☁️ Questions stored in Firebase Firestore
- 🔄 Real-time question fetching
- 🧾 Instant result calculation
- 📊 Final score display
- ✅ View correct answers after submission
- 🎯 Smooth and user-friendly UI

---

## 🛠️ Built With

- Flutter (Dart)
- Firebase Firestore
- Firebase Core
- Material Design UI

---

## ☁️ Firebase Integration

This app uses Firebase as the backend:

- **Cloud Firestore**
  - Stores quiz questions
  - Stores options and correct answers

- **Firebase Core**
  - Initializes Firebase connection in Flutter app
